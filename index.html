Perceptron Model Implementation Flow

1. Initialization

Create the Perceptron instance. This sets up the initial state.

  • Initialize Weights (w): A vector of random values (e.g., between -1 and 1). One weight for each input feature.
    self.weights = 2 * np.random.rand(input_size) - 1
  • Initialize Bias (b): A single random value.
    self.bias = np.random.rand(1)
  • Set Learning Rate (α): A small constant that controls the step size of adjustments (e.g., 0.1).
    self.learning_rate = learning_rate
↓
2. Training Loop

Iteratively adjust the weights and bias to minimize the error. This is done by calling perceptron.train().

While Average Error > Max Error:

For each training example (inputs, target):

  1. Forward Pass (Predict):
    Calculate the weighted sum and pass it through the activation function (Sigmoid).
    net_input = np.dot(inputs, self.weights) + self.bias
    prediction = self.sigmoid(net_input)

  2. Calculate Error:
    Find the difference between the desired target and the actual prediction.
    error = target_output - prediction

  3. Update Parameters (Backpropagation):
    Adjust weights and bias based on the error and learning rate.
    adjustment = self.learning_rate * error
    self.weights += adjustment * inputs
    self.bias += adjustment

Repeat this process for many epochs (passes through the entire dataset) until the model's average error is acceptably low.

↓
3. Evaluation & Prediction

Once training is finished, use the tuned weights and bias to make predictions on new or existing data.

  • Make Predictions: Pass the test data through the perceptron.predict() method.
  • Apply Threshold: Convert the Sigmoid output (a probability between 0 and 1) into a binary class (0 or 1).
    output = 1 if prediction > 0.5 else 0
  • Calculate Metrics: Compare the final predictions against the true targets to measure performance using tools like Accuracy, Confusion Matrix, etc.
    accuracy = accuracy_score(targets, predictions)
